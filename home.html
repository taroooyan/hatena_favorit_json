概要

はてなブックマークのお気に入りユーザを利用しようとした際に公式のAPIでは提供されていないと思われるため、あるユーザのお気に入りのユーザリストJSON形式で返すものを作成

使い方

'/はてなブックマークのユーザ名' にアクセスすることで取得できる。 
    
 フォーマット 

    [
        {
            "profile-image": "http://cdn1.www.st-hatena.com/users/******1/profile_l.gif",
            "username": "*****1"
        }, 
        {
            "profile-image": "http://cdn1.www.st-hatena.com/users/******2/profile_l.gif",
            "username": "*****2"
        }
    ]
